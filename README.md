**Problem Statement**

Faculty often struggle to create diverse question banks on tight schedules, and there’s no easy way to turn PDFs into quizzes. Students taking quizzes asynchronously can reduce fairness and engagement. Existing tools also offer limited feedback and tracking. This highlights the need for a smart, centralized system to tackle all these challenges.

Smart Quiz Generator — an AI-powered web app that converts PDFs into MCQs. Faculty and students have separate logins. Teachers can create quizzes manually or using AI, review questions, generate codes, and track class performance. Students join using a quiz code, take timed, synchronized quizzes with a live leaderboard, and see their personal performance reports afterward.
